## 1. 简介
Sound PNG 是一款专业的隐写工具，利用 LSB 算法将文件隐藏在图片或音频中，而不破坏载体的正常使用。v1.3.1 版本引入了全新的通用流式架构和插件系统。

---

## 2. 界面概览
软件主界面包含四个主要标签页：
- **Encode (编码)**: 执行隐写操作。
- **Decode (解码)**: 提取隐藏文件。
- **Batch Mode (批量模式)**: 批量处理多个文件。
- **Settings (设置)**: 语言、主题、插件管理和开发者选项。

底部状态栏显示当前操作进度和状态信息。

---

## 3. 基础操作

### 编码 (隐藏文件)
1. **Payload (负载)**: 点击 "Browse" 选择你要隐藏的文件（任意格式）。
2. **Container (容器)**: 选择一张 PNG 图片或 WAV 音频。
3. **Save As**: 指定输出文件的保存位置。
4. **Encryption (加密)**: (可选) 勾选并选择一个文件作为密钥。
5. 点击 **Encode** 按钮。如果负载过大，系统会弹出警告。

### 解码 (提取文件)
1. **Input (输入)**: 选择含有隐写数据的 PNG/WAV 文件。
2. **Key (密钥)**: 如果编码时使用了密钥，此处必须提供相同的文件。
3. **Output**:
   - **Preset (预设)**: 可选择提取后文件的扩展名（如 PNG, ZIP, MP4 等）。
   - **Resize (缩放)**: 仅对提取出的图片有效，可调整尺寸。
4. 点击 **Decode**。

## 4. 高级功能

### 批量处理
在 **Batch Mode** 标签页中：
- **Batch Encode**: 添加多个负载文件，选择一个统一的容器。程序将为每个负载生成一个独立的隐写文件。
- **Batch Decode**: 添加多个隐写文件，程序将批量提取内容到指定目录。

### 序列帧模式 (插件)
当启用 **Sequence Frame Plugin** 后：
- 在编码/解码页面勾选 **Sequence Mode (Folder)**。
- **Container**: 选择一个包含序列帧图片（如 `001.png`, `002.png`...）的文件夹。
- 程序会将大文件分散隐藏到该文件夹的所有图片中。

### Python 插件
允许调用外部 Python 脚本进行处理：
1. 在设置中启用插件。
2. 配置 Python 路径（若未自动检测到）。
3. 编码/解码时，任务将通过本地端口 (默认 6657) 转发给 Python 进程。
4. 一个独立的命令行窗口将显示 Python 端的运行日志。

## 5. 设置与配置
- **Language**: 切换中文/英文界面。
- **Dark Mode**: 切换深色/浅色主题。
- **Stream Buffer**: 调整流式处理的缓冲区大小（默认 64KB）。对于机械硬盘或网络驱动器，增加缓冲区可能提高性能。
- **Dev Mode**: 开启后，界面底部将显示日志控制台。

---

## 6. 常见问题
- **Q: 解码失败或文件损坏？**
  - A: 请确认密钥文件完全一致。如果是图片载体，请确保图片未经过二次压缩（如微信/QQ传输会破坏隐写数据）。
- **Q: 找不到 Python？**
  - A: 请在设置中手动指定 `python.exe` 的完整路径。
